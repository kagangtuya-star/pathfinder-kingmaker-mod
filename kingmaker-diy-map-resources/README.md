使用时请阅读[地图注释 |铸造虚拟桌面 (foundryvtt.com)](https://foundryvtt.com/article/map-notes/)的官方教程，使用权限控制获得最佳探索体验（即给pc已知的地方日志给予拥有权限，方便他们自行修改等）
